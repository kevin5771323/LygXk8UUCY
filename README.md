# 共享汽车管理系统简介 java30

## 项目概述

共享汽车管理系统是基于Springboot+Vue的前后端分离的Javaee项目。项目采用MVC三层架构，整合Spring、SpringBoot、Mybatis和Vue等技术开发，运用MySQL数据库进行数据存储，实现了便捷的Web操作界面。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 项目构建：Maven
- 架构：MVC

## 功能模块

- 用户管理
- 车辆管理
- 订单管理
- 费用结算
- 数据统计

## 系统角色

- 系统管理员
- 普通用户
- 维护人员

## 运行环境

- JDK 1.8+
- MySQL 5.7+
- Maven 3.5+

## 部署步骤

1. 准备好运行环境
2. 通过Maven构建项目
3. 部署应用到服务器
4. 启动后端服务
5. 运行前端Vue项目

## 目录结构

```
├── frontend      // 前端Vue项目
└── backend       // 后端Springboot项目
    ├── src
    ├── config
    ├── sql        // 数据库脚本
    └── pom.xml    // Maven项目文件
```
